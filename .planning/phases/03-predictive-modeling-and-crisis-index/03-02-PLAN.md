  <action>
1. Create `backend/ecpm/modeling/stationarity.py`:
   - `test_stationarity(series: pd.Series, alpha: float = 0.05) -> dict` -- dual ADF + KPSS test. ADF null = unit root (non-stationary), KPSS null = stationary. Return dict with adf_pvalue, kpss_pvalue, adf_stationary, kpss_stationary, is_stationary (both agree), recommendation ("stationary" or "difference"). When tests disagree, default to "difference" (conservative). Use regression='c', autolag='AIC' for ADF; regression='c', nlags='auto' for KPSS. Suppress the FutureWarning from kpss using warnings.catch_warnings.
   - `ensure_stationarity(data: pd.DataFrame) -> tuple[pd.DataFrame, dict[str, int]]` -- test each column. If ANY column is non-stationary, difference ALL columns (VAR requires equal-length series). Return (transformed_data, diff_orders). Log results with structlog.

2. Create `backend/ecpm/modeling/var_model.py`:
   - `fit_var_model(data: pd.DataFrame, max_lags: int = 12) -> dict` -- create VAR(data), call select_order(max_lags), use AIC-optimal lag (with BIC as tiebreaker if AIC returns 0). Fit model. Return {"results": results, "selected_lag": int, "aic": float, "bic": float, "hqic": float, "lag_criteria": dict}. Log lag selection with structlog.
   - `generate_forecasts(results, data: pd.DataFrame, steps: int = 8) -> dict[str, dict]` -- extract last lag_order observations, call forecast_interval with alpha=0.32 (68% CI) and alpha=0.05 (95% CI). Return per-column dict with point/lower_68/upper_68/lower_95/upper_95 as Python lists (not numpy arrays). Handle the inverse differencing: accept optional diff_orders param; if provided and any order > 0, cumsum the forecasts back to levels using the last observed value as base.

3. Create `backend/ecpm/modeling/svar_model.py`:
   - `build_svar_a_matrix(n_vars: int = 8) -> np.ndarray` -- construct lower-triangular A matrix per Marxist causal ordering: (1) OCC, (2) rate of surplus value, (3) rate of profit, (4) mass of profit, (5) productivity-wage gap, (6) credit-GDP gap, (7) financial-real ratio, (8) debt service ratio. Diagonal = 1, below diagonal = 'E', above = 0. Use dtype=object for mixed types.
   - `fit_svar(var_results, a_matrix: np.ndarray) -> object` -- create SVAR model from the original endogenous data with svar_type='A', fit with maxlags from var_results.k_ar. Return svar_results. Wrap in try/except: if SVAR fitting fails (common with real data), log warning and return None.

All modules use structlog for logging. All functions are pure (no side effects beyond logging).

Extract smoothed_marginal_probabilities, current_regime (argmax of last row), regime_transition matrix. Map regime indices to labels by mean of each regime (highest mean = "Normal", middle = "Stagnation", lowest = "Crisis"). If k_regimes=3 fails to converge (ConvergenceWarning or RuntimeError), retry with k_regimes=2 (Normal/Crisis only). Log which was used. INVERT rate_of_profit (lower = higher crisis) and rate_of_surplus_value (lower = higher crisis). For OCC and mass_of_profit, KEEP direction (higher OCC = higher crisis). Average within each mechanism group. Compute weighted composite * 100 for 0-100% scale. Return {"composite": pd.Series, "trpf": pd.Series, "realization": pd.Series, "financial": pd.Series}. Handle missing indicators gracefully (skip unavailable, weight from available).
