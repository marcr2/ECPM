<!-- From Plan 03 -- API endpoints the frontend connects to -->

Backend API endpoints:
```
GET  /api/data/series          -> { series: [...], total: N }
  Query params: source, status, search, sort_by, sort_order

GET  /api/data/series/{id}     -> { metadata: {...}, observations: [...] }
  Query params: start_date, end_date, frequency, fill_method

GET  /api/data/status          -> { last_fetch_time, next_scheduled_run, total_series, ok_count, error_count, stale_count, errors: [...] }

POST /api/data/fetch           -> { task_id, status: "submitted" }

GET  /api/data/fetch/stream    -> SSE events (start, progress, complete)

GET  /health                   -> { status: "healthy" }
```

Response types:
```typescript
interface SeriesMetadata {
  series_id: string;
  source: "FRED" | "BEA";
  name: string;
  units: string | null;
  frequency: "D" | "M" | "Q" | "A";
  seasonal_adjustment: string | null;
  last_updated: string | null;  // ISO datetime
  last_fetched: string | null;
  observation_count: number;
  fetch_status: "pending" | "ok" | "stale" | "error";
  fetch_error: string | null;
}

interface FetchStatus {
  last_fetch_time: string | null;
  next_scheduled_run: string | null;
  total_series: number;
  ok_count: number;
  error_count: number;
  stale_count: number;
  errors: Array<{ series_id: string; error_message: string }>;
}
```
