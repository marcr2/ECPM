         - `seasonal_adjustment`: String(50), nullable
         - `last_updated`: DateTime(timezone=True), nullable -- when source last updated
         - `last_fetched`: DateTime(timezone=True), nullable -- when we last fetched
         - `observation_count`: Integer, default 0
         - `fetch_status`: String(20), default "pending" -- pending/ok/stale/error
         - `fetch_error`: Text, nullable
         - `created_at`: DateTime(timezone=True), server_default=func.now()
