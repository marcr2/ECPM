**crisis-annotations.tsx** -- Per 02-RESEARCH.md Pattern 3:
- Props: `crises: CrisisEpisode[]`, `visible: 'shaded' | 'lines' | 'hidden'`
- When 'shaded': render `<ReferenceArea>` for each crisis with x1/x2/fill/fillOpacity(0.15)
- When 'lines': render two `<ReferenceLine>` per crisis (start + end) with dashed stroke
- When 'hidden': return null
- Must be `'use client'`

**indicator-chart.tsx** -- Main chart component per 02-RESEARCH.md Pattern 3:
- Props: `data: { date: string; [key: string]: number | string }[]`, `primaryKey: string`, `primaryLabel: string`, `overlayKey?: string`, `overlayLabel?: string`, `crisisMode: 'shaded' | 'lines' | 'hidden'`, `height?: number`
- Uses Recharts `ComposedChart` (NOT LineChart) inside `ResponsiveContainer`
- `CartesianGrid` with strokeDasharray="3 3" and `stroke="hsl(var(--border))"`
- `XAxis` with `dataKey="date"`, formatted as year
- `YAxis` yAxisId="left" for primary indicator
- Optional `YAxis` yAxisId="right" for overlay (dual y-axis per user decision)
- `Line` for primary, optional `Line` for overlay
- `Tooltip` and `Legend`
- `<CrisisAnnotations>` rendered inside the chart
- `<Brush>` component at bottom with `dataKey="date"` and `height={30}` for zoom/pan (per user decision)
- Use Recharts 3.x API -- do NOT use activeIndex, CategoricalChartState, or blendStroke (removed in 3.x)
- `'use client'` component

**sparkline.tsx** -- Per 02-RESEARCH.md Sparkline Pattern:
- Props: `data: number[]`, `color?: string`, `height?: number`
- Mini `LineChart` in `ResponsiveContainer`, no axes, no tooltip, no grid
- `dot={false}`, `animationDuration={0}`

**date-range-controls.tsx** -- Preset buttons:
- Props: `onRangeChange: (years: number | null) => void`, `activeRange: number | null`
- Buttons: 5Y, 10Y, 25Y, 50Y, All (null = all)

- `'use client'`

**overlay-selector.tsx** -- "Add overlay" dropdown:

- Can use a basic `<select>` styled with Tailwind or a shadcn Select if installed
